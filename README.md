# Jimenez Mobile Detailing

Premium, mobile-first static website for Jimenez Mobile Detailing in the Paso Robles area.

## Pages

- `index.html`
- `about/index.html`
- `before-after/index.html`

## Local Preview

Open `index.html` directly in a browser, or run a simple static server from this folder.

## EmailJS

The inquiry form uses EmailJS so visitors stay on the site and no email app opens.

The EmailJS setup is in `src/app.js`:

```txt
service_uf0e37g
template_m7zwpf9
z2UjAu1N-IFmF2pbc
```

The EmailJS template expects:

```txt
from_name
phone
vehicle_info
service_location
selected_package
preferred_date
preferred_time
addons
notes
reply_to
```
